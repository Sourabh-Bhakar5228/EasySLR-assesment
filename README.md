# EasySLR - Article Review Workspace

EasySLR is a professional-grade Article Review Workspace application designed to conduct Systematic Literature Reviews (SLRs) efficiently. This repository contains the complete **Day 1** setup foundation.

---

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (v3)
- **Database ORM**: Prisma Client
- **Database Engine**: PostgreSQL
- **Authentication**: NextAuth.js (Credentials Provider)
- **Forms Management**: React Hook Form
- **Validation**: Zod
- **Icons**: Lucide React
- **Notifications**: Sonner

## 🚀 Setup & Local Installation

### 1. Prerequisite Installations
Ensure you have the following installed on your machine:
- Node.js (v18.x or above recommended)
- PostgreSQL (Active and running locally or hosted)

### 2. Configure Environment Variables
Copy `.env.example` to `.env` and fill in the database URL and credentials secret:
```bash
cp .env.example .env
```
Ensure your `DATABASE_URL` is pointing to an active PostgreSQL database:
```env
DATABASE_URL="postgresql://postgres:postgres@localhost:5432/easyslr?schema=public"
NEXTAUTH_SECRET="easy-slr-secret-jwt-key-minimum-32-characters-long"
NEXTAUTH_URL="http://localhost:3000"
```

### 3. Install Dependencies
Run npm install:
```bash
npm install
```

### 4. Database Setup (Migrations & Seeding)
Apply Prisma database migrations to create your schemas and tables:
```bash
npx prisma migrate dev --name init
```

Seed the database with default organizations, projects, and users:
```bash
npx prisma db seed
```

### 5. Run the Application
Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
